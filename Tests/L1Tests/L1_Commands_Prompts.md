# L1 Commands, Prompts

## Commands (Mac)

```bash
brew install act
brew install --cask docker
```

- for mac, local folders are not binding though we give `--bind`; copy required folders to container using:

```bash
docker cp <local_folder_path> <container_name>:<destination_path>
```

## Commands (Ubuntu)

```bash
curl https://raw.githubusercontent.com/nektos/act/master/install.sh | sudo bash
sudo apt update
sudo apt -y install docker.io
```
